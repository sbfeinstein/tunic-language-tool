# Tunic Language Tool

[![GitHub Pages Deployment](https://github.com/sbfeinstein/tunic-language-tool/actions/workflows/pages/pages-build-deployment/badge.svg)](https://sbfeinstein.github.io/tunic-language-tool/)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

Tunic Language Tool (TLT) is a fan-created project related to the secret language from the 
award-winning game [Tunic](https://tunicgame.com/)!

You can learn all about TLT and use it online for free on the [Tunic Language Tool Site](https://sbfeinstein.github.io/tunic-language-tool/).

TLT is free, open-source software released under the [MIT License](LICENSE). 

It is a tool to help with solving the mystery of the in-game runic language.

TLT is not affiliated with Isometricorp Games LTD, Finji or any other official entity related to Tunic the game.

# Technical details
TLT is a Vue3 application and you can refer to the *VueJS Autogenerated Docs* below for more information about VueJS.

The excellent [Tiptap](https://tiptap.dev/) headless editor platform is a core project dependency.

The static Tunic Language Tool Site is built using the wonderful [Zensical](https://zensical.org/) framework.

TLT has no server-side component, it runs entirely in the browser.  Additionally, it does not make use of any ads, 
authentication or tracking technologies.

# Running locally

## Prerequisites

To run this project locally, you must install these two system-level dependencies.

1. Node Version Manager(nvm)
    * **macOS/Linux:** [nvm](https://github.com/nvm-sh/nvm)
    * **Windows:** [nvm-windows](https://github.com/coreybutler/nvm-windows)
2. [uv](https://astral.sh/uv)
    * **macOS/Linux:** `curl -LsSf https://astral.sh/uv/install.sh | sh`
    * **Windows:** `powershell -c "irm https://astral.sh/uv/install.ps1 | iex"`

## Building and running the project

```sh
npm install
npm run build
npm run dev
```

If all went well, you'll be able to browse to:

- http://localhost:8000/tunic-language-tool for the static TLT site, powered by Zensical
- http://localhost:5173/tunic-language-tool/app for the TLT application, powered by Vue

## Deploying the project

The TLT site and application are automatically deployed on every merge to the `main` branch, via its 
[GitHub Actions Workflow](.github/workflows/deploy.yml).

Additionally, there is a manual [deploy script](scripts/deploy.sh) as a fallback mechanism.
