# Tunic Language Tool
This is a fan-created project for the game [TUNIC](https://tunicgame.com/).

It is a tool to help with solving the mystery of the in-game runic language.

There is no relationship between this software and TUNIC the game, its creators or publishers, including FINJI and related entities.

# About the tool
For now this project is strictly in the scratchpad / proto-typing phase.  Stay tuned!
