# Spoilers

TLT is a Tunic document editor and while it does not directly provide the "solution" to translating the language, 
the nature of its user interface includes several spoilers.  

We highly recommend you do not use TLT too early in your game progression!  The game is too good for that!

Here are the potential spoilers within the TLT interface, presented in the order you'd likely encounter them in-game.
As you expand the sections and reveal the spoilers, if you encounter anything you didn't know please **stop**!
Play the game some more and come back to TLT later on.

??? note "Spoiler #1"

    Pages 21 and 54 of the manual (stop here if you haven't found them yet).

??? note "Spoiler #2"

    The game language symbols are basically subsets of lines from a gem-like shape.

    TLT refers to these as "runes".

??? note "Spoiler #3"

    The symbols are divided into two groups representing outer and inner lines.

    TLT refers to these as "Outer Runes" and "Inner Runes".

??? note "Spoiler #4"

    The symbols are combined and drawn on top of each other.

    Valid combinations, using TLT lingo, are:
    
    - Exactly one Outer Rune plus exactly one Inner Rune
    - Exactly one Outer Rune with no Inner Rune
    - Exactly one Inner Rune with no Outer Rune

??? note "Spoiler #5"

    The circle symbol sometimes rendered beneath the runes reverses the order in which they are read.
