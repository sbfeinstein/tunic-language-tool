# Overview

**Tunic Language Tool (TLT)** is a fan-made web app related to the game Tunic.  

!!! info "Where are the screenshots?"
    To avoid potential spoilers there are no screenshots on this page!  
    See [Spoilers](Spoilers.md) for more info!


It is a tool that helps you translate the in-game language but it does *not* directly translate the language for you.
Instead, TLT lets you write in the game's language and test out your translation ideas
(more quickly and easily than using paper and pencil)! 

TLT is **not** affiliated with Isometricorp Games LTD, Finji or any other official entity.

Learn more about the [Features](Features.md) or [try it yourself](TryIt.md)!

TLT has **no ads**, **no logins**, **no subscriptions**, **no tracking** or other annoyances! 
___

[Tunic](https://tunicgame.com/) is an award-winning, thought-provoking action / puzzler game!  
A big part of the fun is unraveling the mystery of the game's mechanics, lore, goals and secret language.
