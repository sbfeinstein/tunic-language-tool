# Tunic (a great game!)
The game [Tunic](https://tunicgame.com/) is an excellent, thought-provoking action / puzzler where a big part of the fun is unraveling the mystery of the game itself.

The rules, goal(s) and even gameplay mechanics are largely a mystery. Even the in-game instruction manual, which is slowly revealed, is largely written in a made-up language.

# Tunic Language Tool!
The vision for this project, the Tunic Language Tool (TLT), is to provide an aid for deciphering the in-game language.

# TLT Architecture
This is a pure [Vue.js](https://vuejs.org/) Single Page Application (SPA).

State is held in memory, with options to import/export JSON so that progress can be saved.
By providing import/export, we also allow the user to explore different theories for how to translate the in-game runes to English and not lose that work.
